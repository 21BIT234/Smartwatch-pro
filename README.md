# SmartWatch Pro Landing Page

A responsive landing page for a fictional product called **SmartWatch Pro**, built using **HTML5** and **CSS3** without JavaScript.

## Project Overview

This project demonstrates core front-end development skills including:

* Semantic HTML
* CSS Grid Layout
* Flexbox Layout
* Responsive Design
* Mobile-First Principles
* Reusable CSS Components

The page contains:

* Hero Section
* Features Grid
* Testimonials Section
* Call-to-Action Footer

## Features

### Hero Section

* Product title and tagline
* Call-to-action button

### Features Grid

* Built using CSS Grid
* Responsive card layout
* Product feature highlights

### Testimonials Section

* Built using Flexbox
* Customer review cards
* Responsive alignment

### Call-to-Action Footer

* Encourages user engagement
* Prominent action button

## Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Media Queries

## Responsive Design

The layout automatically adapts to different screen sizes:

* Desktop View

* Mobile View

1. Clone the repository.
